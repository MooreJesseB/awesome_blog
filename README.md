awesome_blog--a work in progress
============

Microblog weekend lab

To run app: 
* clone this repo locally
* run $bundle install
* $rake db:create
* $rake db:migrate
* $rake db:seed
* then you can work in the $rails console
* OR run $rails server
* open http://localhost:3000/ in your browser


Styled with Semantic UI
